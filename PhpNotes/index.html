Fixing npm's standard Windows security restriction Open the Start Menu, search for PowerShell, right-click it, and select Run as Administrator. Run the following command to allow scripts for your user account: powershell C:\Users\rohid> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser When prompted, type Y (or A for Yes to All) and press Enter.
